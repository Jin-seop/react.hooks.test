# 데모서버 실행하기
React환경이 만들어져 있다면 직접 clone해서 테스트하고, 실행할 수 있습니다.

1. Github에서 clone해오기
```
git clone https://github.com/ninanung/react.hooks.test.git
```

2. 의존 모듈 설치하기
```
cd react.hooks.test
npm install
```

3. 데모서버 실행
```
npm start
```

4. http://localhost:3000포트로 접속해서 동작확인
